Appointment Manager

Version: 0.1.0
Languages: C, C++
C++ Standard: C++17
Database: SQLite3

Overview

The Appointment Manager is a software application designed to manage appointments for a medical center. It provides functionalities for creating, editing, and canceling appointments. Patients' information, including appointment details, are stored in a database, and users can view and manage appointment schedules effectively.

Features

Appointment Management:
Add new appointments
Edit existing appointments
Cancel appointments
Schedule Viewing:
View appointment schedules
Handle calendar conflicts
Patient Management:
Add and manage patient details
Database Integration:
All data stored in an SQLite3 database
Technical Stack

Programming Language: C++17
Database: SQLite3
GUI (Future): Qt (will be integrated in future versions)
Build System: CMake
